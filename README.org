* Description
~coplot.m~ is blilliant plot tool for MATLAB.

Get MATLAB's beautiful plot figures from data csv.

* Install
#+begin_src sh
  git clone https://github.com/conao3/coplot.m.git
#+end_src

* Usage
** basic
Make 'Tab' separeted csv and some configures in 'same' csv file.
